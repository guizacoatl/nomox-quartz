# Article 23 - Accès aux données de marché différées


Les opérateurs de marché et les entreprises d’investissement exploitant une plate-forme de négociation, les APA et les internalisateurs systématiques donnent accès aux données de marché différées à tous les clients sur une base non discriminatoire, sans exiger aucun type d’enregistrement.
