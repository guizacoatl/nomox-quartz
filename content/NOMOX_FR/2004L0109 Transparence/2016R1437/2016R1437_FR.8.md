# Article 8 - Modèle commun pour la fourniture des métadonnées

## 1.
Chaque mécanisme officiellement désigné utilise un format basé sur le langage XML (extensible markup language) pour fournir les métadonnées sur les informations réglementées au PAEE.
## 2.
Chaque mécanisme officiellement désigné communique les métadonnées sur les informations réglementées au PAEE dans le format prévu à la section A de l'annexe.
