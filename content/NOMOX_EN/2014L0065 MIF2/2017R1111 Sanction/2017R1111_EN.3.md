# Article 3 - Invalidating and updating of reports

## 1.
Where a competent authority wishes to invalidate an existing report file it has previously submitted to ESMA in accordance with [[2017R1111_EN.2|Article 2]], it shall cancel the existing report and send a new report file.
## 2.
Where a competent authority wishes to update an existing report file it has previously submitted to ESMA in accordance with [[2017R1111_EN.2|Article 2]], it shall resubmit the report file with the updated information.
